# Calculus Tool

## *О проекте*
Calculus Tool — это расширенная консольная утилита для математических вычислений. Проект поддерживает как базовую арифметику, так и элементы математического анализа (символьное дифференцирование).

## *Функциональность*
- **Арифметика**: `+`, `-`, `*`, `/`, `^`.
- **Алгебра**: Решение квадратных уравнений.
- **Матанализ**: Вычисление производных функций любой сложности (например, `x**2 + sin(x)`).

## *Требования*
- Python 3.8+
- Библиотека `sympy` (см. установку)

## *Установка и запуск (Обязательно!)*

1. **Клонирование:**
   ```bash
   git clone https://github.com/ВАШ_НИКНЕЙМ/calculus_tool.git
   cd calculus_tool
   ```

2. **Создание виртуального окружения:**
   ```bash
   # Windows
   python -m venv venv
   venv\Scripts\activate

   # macOS/Linux
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Установка зависимостей (ВАЖНО):**
   Без этого шага программа выдаст ошибку, так как используется библиотека `sympy`.
   ```bash
   pip install -r requirements.txt
   ```

4. **Запуск:**
   ```bash
   python src/main.py
   ```
